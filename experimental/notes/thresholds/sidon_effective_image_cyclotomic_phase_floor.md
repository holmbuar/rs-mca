# Trace-linear cyclotomic phase floor

## Result and route cut

This packet proves `COUNTEREXAMPLE_NEW_FLOOR`, acceptance criterion **4**.
The family uses the genuine degree-one Reed--Solomon/Vandermonde phase
`g(t)=t`; phases are `p`-th roots generated by the trace pairing, not assigned
signs. A canonical balanced phase-histogram block has identical positive
Fourier coefficients, so cancellation inside the block is exactly zero.

```text
route cut: PHASE_HISTOGRAM_LOCAL_MI_MA
successor: ACTUAL_LEAF_CROSS_HISTOGRAM_CANCELLATION_OR_DIRECT_SIDON
```

Thus “use RS phases” is not enough. A surviving method must couple different
histograms through the target character, or bypass MI+MA with a direct
image-normalized Sidon/max-fiber theorem on the actual post-C1--C8 residual.

## Construction

Fix an odd prime `p` and `r>=1`. Put `N=2pr`, `m=pr`, and
`K=F_(p^(N-2))`. Choose an `F_p`-basis `e_1,...,e_(N-2)` of `K` and set

```text
u = -(e_1+...+e_(m-1)),
T = {0,e_1,...,e_(N-2),u},
S_* = {e_1,...,e_(m-1),u},
Psi(S) = sum_(t in S) t,  S in binom(T,m).
```

This is the first power-sum coordinate of `eq:exact-power-sum-map`.
Characters are the actual trace-linear phases
`chi_a(t)=zeta_p^Tr_(K/F_p)(at)`.

The differences from `0` span `K`, hence `A_eff=p^(N-2)`. The map `Psi` is
injective on the `m`-slice. If two supports collide and their incidence
difference at `u` is `c in {-1,0,1}`, basis comparison forces difference `c`
on `e_1,...,e_(m-1)` and zero elsewhere. Equal cardinality then requires the
difference at `0` to be `-cm`, impossible for `c!=0` because `m>=3`.
Therefore

```text
M = L = binom(N,m),
Psi^(-1)(0) = {S_*}.
```

The trace pairing parametrizes phase words by arbitrary values on the basis,
with the single relation `y_u=-(y_1+...+y_(m-1))`.

## Coherent cyclotomic block

Let each residue of `F_p` occur exactly `r` times on `S_*` and exactly `r`
times on its complement. Put `H_(p,r)=(pr)!/(r!)^p`. Every balanced word on
`S_*` satisfies the relation because `r(0+...+(p-1))=0` in `F_p`; there are
`H_(p,r)` choices. Fixing the base phase to zero leaves `H_(p,r)/p` choices on
the complement. Thus

```text
B_(p,r) = H_(p,r)^2/p.                              (1)
```

Every block character has `2r` copies of each phase. For odd `p`,

```text
prod_(j=0)^(p-1)(1+zeta_p^j z) = 1+z^p,
```

so its coefficient is

```text
[z^(pr)](1+z^p)^(2r) = binom(2r,r) > 0.             (2)
```

The target is zero, so all target factors are one. The signed block sum is
therefore coherent:

```text
A_phase(p,r) = (H_(p,r)^2/p) binom(2r,r).           (3)
```

This is not an absolute-value relaxation.

## General family theorem

The balanced multinomial is maximal among at most `(pr+1)^(p-1)` types, and
the central binomial coefficient is maximal among `2r+1` terms. Hence

```text
H_(p,r) >= p^(pr)/(pr+1)^(p-1),
binom(2r,r) >= 2^(2r)/(2r+1).
```

Since `L=M`, exact image compensation gives

```text
(L/A_eff)(A_phase/M) = A_phase/A_eff
 >= p*2^(2r)/((pr+1)^(2(p-1))*(2r+1))              (4)
 = exp((log 2)N/p - O_p(log N)).
```

Thus every fixed odd `p` leaves an exponential compensated block floor.
Also `M<=2^N`, so

```text
A_phase/M >= p^(-1)(p/2)^N/(pr+1)^(2(p-1)).        (5)
```

For fixed `r>=2` and odd primes `p->infinity`, (5) is
`exp(Omega_r(N log N))`: the raw phase-aware block cost is super-exponential.
